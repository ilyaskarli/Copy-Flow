# Copy Flow

A VS Code extension to copy file path and line numbers automatically on selection.


<img width="657" height="341" alt="image" src="https://github.com/user-attachments/assets/6f218944-ffc2-49ae-9727-c05ea758ab06" />

## 🚀 How to Enable Auto-Copy

1. Open VS Code Settings (`Ctrl + ,`).
2. Search for **`Copy-flow: Auto Copy On Selection Change`**.
3. Check/enable the option.

Now, whenever you select text in the editor, the file path and line numbers will be automatically copied to your clipboard.

## ⌨️ Keyboard Shortcut

Alternatively, you can copy manually using the shortcut:
- **Windows/Linux**: `Alt + C`
- **Mac**: `Cmd + Alt + C`
